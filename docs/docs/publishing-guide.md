# 微信公众号发文标准流程（固定版）

> 从选题到发布的完整 7 步流程，包含技能清单、快速命令参考、常见问题 FAQ。

---

## 技能清单

### 内容创作技能
- **选题策划**: 热点分析、用户需求调研
- **文案撰写**: 文章、标题、摘要生成
- **内容优化**: 错别字检测、人性化改写

### 配图生成技能
- **AI 配图**: 根据文章内容自动生成配图
- **图片优化**: 裁剪、调色、水印添加

### 发布技能
- **HTML 排版**: Markdown 转 HTML，卡片式排版
- **草稿箱保存**: 自动保存到公众号草稿箱
- **定时发布**: 设置定时任务，自动发布

---

## 标准 7 步流程

### 第 1 步：选题策划（30 分钟）
**目标**: 确定今天要写什么

**动作**:
1. 分析热点（微博热搜、知乎热榜、行业新闻）
2. 调研用户需求（留言、评论、私信）
3. 确定选题方向（技术科普/行业案例/产品更新）

**产出**: 选题文档（包含标题、大纲、核心观点）

---

### 第 2 步：内容创作（1-2 小时）
**目标**: 完成文章初稿

**动作**:
1. 告诉 AI 选题和大纲
2. AI 生成文章初稿
3. 人工审核和修改

**产出**: 文章初稿（Markdown 格式）

**示例命令**:
```
帮我写一篇关于 OpenClaw 的技术科普文章
主题：普通人的 AI 超级助手
大纲：
1. 什么是 OpenClaw
2. 能做什么
3. 真实案例
4. 新手如何开始
字数：3000 字左右
风格：口语化，像跟朋友聊天
```

---

### 第 3 步：内容检测（10 分钟）
**目标**: 确保内容合规、无错误

**动作**:
1. 错别字检测
2. 敏感词检测
3. 逻辑检查

**产出**: 检测报告 + 修改建议

**示例命令**:
```
检查这篇文章的错别字和敏感词：
[粘贴文章内容]
```

---

### 第 4 步：人性化改写（20 分钟）
**目标**: 让文章更像人写的，不机械

**动作**:
1. 添加个人经历/案例
2. 调整语气（更口语化）
3. 增加互动元素（提问、引导评论）

**产出**: 优化后的文章

**示例命令**:
```
把这篇文章改写得更加口语化，像跟朋友聊天：
[粘贴文章内容]
要求：
- 多用"你""我"对话感
- 加入真实案例
- 增加互动提问
```

---

### 第 5 步：AI 配图（10 分钟）
**目标**: 生成文章专属配图

**动作**:
1. 提取文章关键词
2. 调用 AI 配图 API
3. 下载并保存配图

**产出**: 3-5 张高清配图

**示例命令**:
```
根据这篇文章内容，生成 3 张配图：
文章主题：OpenClaw 介绍
风格：科技感、简洁、现代
尺寸：1280x720
```

---

### 第 6 步：HTML 排版（15 分钟）
**目标**: 将 Markdown 转换为精美的 HTML

**动作**:
1. Markdown 转 HTML
2. 添加样式（卡片式、代码高亮）
3. 插入配图

**产出**: 排版好的 HTML 内容

**示例命令**:
```
把这篇文章转换为 HTML，要求：
- 卡片式排版
- 代码块高亮
- 插入配图（路径：./images/）
- 添加关注公众号的引导卡片
```

---

### 第 7 步：发布到草稿箱（5 分钟）
**目标**: 保存到公众号草稿箱，准备发布

**动作**:
1. 调用公众号 API
2. 上传图文内容
3. 保存到草稿箱

**产出**: 草稿箱链接 + 预览二维码

**示例命令**:
```
发布这篇文章到公众号草稿箱：
标题：OpenClaw：普通人的 AI 超级助手
封面图：./images/cover.jpg
内容：[HTML 内容]
```

## 快速命令参考

### 内容创作
```bash
# 生成文章
openclaw write --topic "选题" --outline "大纲" --words 3000

# 检查错别字
openclaw check --spell article.md

# 人性化改写
openclaw rewrite --style conversational article.md
```

### 配图生成
```bash
# 生成配图
openclaw images --count 3 --style tech article.md

# 下载配图
openclaw download --images --output ./images/
```

### 排版发布
```bash
# HTML 排版
openclaw format --style card article.md

# 发布到草稿箱
openclaw publish --platform wechat --draft article.html
```

## 常见问题 FAQ

### Q1：选题没灵感怎么办？
**A**: 用 AI 帮你 brainstorm。告诉 AI 你的目标用户和领域，让它生成 10 个选题建议，然后选最感兴趣的。

### Q2：文章写得太长/太短怎么办？
**A**: 在生成时明确字数要求。如果初稿不合适，可以让 AI"精简到 2000 字"或"扩展到 5000 字"。

### Q3：配图不满意怎么办？
**A**: 调整提示词，描述更具体的画面。或者手动选择图片，AI 只负责生成灵感。

### Q4：发布失败怎么办？
**A**: 检查公众号 API 权限、access_token 是否过期。查看错误日志，常见问题在技能包 README 里有解决方案。

### Q5：如何定时发布？
**A**: 当前版本需要手动在公众号后台设置定时。后续版本会支持自动定时发布功能。
