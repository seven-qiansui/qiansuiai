# 🧠 MEMORY 记忆模型

千岁团队长期记忆规范 v2.0

---

## 核心原则

**MEMORY.md 只记录事实**，不包含规则。

| 内容类型 | 放哪 |
|---------|------|
| 事实/事件 | ✅ MEMORY.md |
| 规则/红线 | ❌ → AGENTS.md |
| 配置/路径 | ❌ → TOOLS.md |

---

## 记录格式

### 人物记录
```
姓名 + open_id + 角色 + 权限
例：用户 544091（老板）— ou_xxx — L0
```

### 事件记录
```
日期 + 事件描述 + 结果
例：2026-03-16 创建安全策略技能，部署三级指令过滤
```
