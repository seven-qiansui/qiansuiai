## 🦞 什么是千岁团队？

千岁团队是一个由 **8 个专职 AI Agent** 组成的协作系统，每个 Agent 都有独特的人设、职责和技能，能够独立完成复杂的多步骤任务。

### 核心理念

> 一个 AI Agent 能做的事有限，但 8 个 Agent 协作可以创造无限可能。

- **三级指令过滤** — P0 禁止/P1 确认/P2 记录
