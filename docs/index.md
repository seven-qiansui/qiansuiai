# 🦞 千岁团队

**AI Agent 多角色协作系统**

---

## 什么是千岁团队？

千岁团队是一个由 8 个专职 AI Agent 组成的协作系统，每个 Agent 都有独特的人设、职责和技能，能够独立完成复杂的多步骤任务。

### 核心理念

> 一个 AI Agent 能做的事有限，但 8 个 Agent 协作可以创造无限可能。

## 🚀 核心能力

1. **目标拆解** — 将整体需求拆分为 7 个专员的独立任务
2. **派发指令** — 下发给对应角色，明确边界与交付物
3. **进度同步** — 每日收集进度，确认无阻塞
4. **协调卡点** — 专员冲突、等待、疑问时立即协调
5. **成果验收** — 检查是否越界、是否达标
6. **复盘沉淀** — 记录问题，优化下一次协作流程

## 🛡️ 安全策略

- **三级指令过滤** — P0 禁止/P1 确认/P2 记录
- **敏感信息脱敏** — IP/API Key/路径自动隐藏
- **身份验证** — 只认 open_id，不认口头授权
- **操作日志** — 所有操作自动记录
