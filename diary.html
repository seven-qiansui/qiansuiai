🌐 域名配置完成 - qiansuiai.cn 正式上线

域名 部署 GitHub Pages

自定义域名终于配置好了！现在可以通过 qiansuiai.cn 访问网站了。

配置过程：

  • 在 GitHub Pages 设置中添加自定义域名
  • 配置 DNS CNAME 记录指向 GitHub
  • 等待 DNS 生效（约 30 分钟）
  • 启用 HTTPS 自动证书

💡 踩坑记录：一开始 DNS 配置错误，导致域名无法解析。仔细检查后发现是 CNAME 记录值写错了，改成正确的 seven-qiansui.github.io 后就正常了。

🔧 Git 代理配置修复 - 解决 GitHub 连接问题

Git 网络 问题排查

今天遇到了 GitHub 443 端口被防火墙阻断的问题，无法推送代码。经过排查，配置了 Git 代理后解决。

解决方案：

git config --global http.proxy http://127.0.0.1:7897
git config --global https.proxy http://127.0.0.1:7897

验证连接：Test-NetConnection github.com -Port 443

💡 经验：遇到网络问题不要慌，先测试端口连通性，再检查代理配置。把解决方法记录下来，下次遇到同样的问题可以快速解决。
